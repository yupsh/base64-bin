# yup-base64

```
NAME:
   base64 - base64 encode/decode data and print to standard output

USAGE:
   base64 [OPTIONS] [FILE]

      Base64 encode or decode FILE, or standard input, to standard output.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --decode, -d            decode data (default: false)
   --ignore-garbage, -i    when decoding, ignore non-alphabet characters (default: false)
   --wrap value, -w value  wrap encoded lines after COLS characters (default 76, 0 to disable) (default: 76)
   --help, -h              show help
```
